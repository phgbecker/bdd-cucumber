# Cucumber Examples

## About

A simple project used to exaplain the Cucumber framework.

## Requirements

- Java JDK 8
- Maven

## How to run

Usage: 
  - Clone this project
  - On path `cucumber-examples` execute the command `mvn clean install -D skipTests`
  - Run all tests with `mvn test`
