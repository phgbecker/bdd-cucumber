## :heavy_exclamation_mark: Requirements

- JDK 8
- Maven
- Google Chrome

## :arrow_forward: How to run

- Clone this project
- On path `selenium-java` execute `mvn install`
- Run all test with `mvn test -Dmaven.test.skip=false`
